Fare calculator for Coastal Care Transport
Website: https://www.coastalcaretransport.com.au
Calculator data stored in Googlesheet 
https://docs.google.com/spreadsheets/d/154dAUAW55ZdVyXRES70qDNGuPXNaYkxliLGbc8pg2uU/edit?gid=1059243316#gid=1059243316
calculator.html provides user with option to select pick up and drop off locations and then calculates the return trip cost
Data in sheet contains matrix and returns distance between two points, the calculator multiplies this by the current per km rate ($1.70) and multiplies by 2 for the return trip
Manifest and Json files added to support Progressive Web App capability
