Fare calculator for Coastal Care Transport<BR>
Website: https://www.coastalcaretransport.com.au<BR>
Calculator data stored in Googlesheet<BR>
https://docs.google.com/spreadsheets/d/154dAUAW55ZdVyXRES70qDNGuPXNaYkxliLGbc8pg2uU/edit?gid=1059243316#gid=1059243316<BR>
calculator.html provides user with option to select pick up and drop off locations and whether single or return trip is required and then calculates the trip cost<BR>
Data in sheet contains matrix and returns distance between two points, the calculator multiplies this by the current per km rate ($1.70) and multiplies by 2 for the return trip<BR>
Fares are rounded UP to nearest 50 cents<BR>
Manifest and Json files added to support Progressive Web App capability
